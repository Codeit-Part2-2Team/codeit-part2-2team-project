## 6. 도메인 주의사항

### 6.1 알약 이미지 특성

- 시각적으로 유사한 클래스 다수 (색·형태 동일, **각인(마킹)** 만 차이)
- 고해상도 이미지에서 작은 각인 검출 필요 → 작은 객체 검출 전략 고려

### 6.2 데이터 증강 주의

| 증강 | 권장 여부 | 이유 |
|------|-----------|------|
| Horizontal Flip | ❌ 사용 금지 | 각인 방향이 뒤집혀 클래스 오판 가능 |
| Vertical Flip | ❌ 사용 금지 | 동일 이유 |
| Color Jitter | ✅ 권장 | 조명 변화 대응 |
| Scale / Crop | ✅ 권장 | 다양한 촬영 거리 대응 |
| Rotation (±15°) | ⚠️ 소량 허용 | 각인 가독성 저하 최소화 |

### 6.3 On-Device Inference 고려

최종 목표: ONNX / TFLite export + 벤치마크 보고서 포함

- 모델 선택 기준에 **파라미터 수 / 추론 속도** 포함
- YOLOv8n 우선 (경량), 필요 시 YOLOv8s 검토
